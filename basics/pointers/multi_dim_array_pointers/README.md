# Pointer to MultiDimensional Array

**i. Decoding Two Dimensional Array:**

```c
            x[2][3] = { {1, 2, 3}, {4, 5, 6} }


            --------- Address of each element -------------
            &x[0][0]=-9648
            &x[0][1]=-9644
            &x[0][2]=-9640
            &x[1][0]=-9636
            &x[1][1]=-9632
            &x[1][2]=-9628
            ------------------------------------------------

            ------------------ Address &x[i] --------------
            &x[0]=-9648
            &x[1]=-9636
            ------------------------------------------------

            ------------------ Content x[i] ---------------
            x[0]=-9648
            x[1]=-9636
            ------------------------------------------------


            ------------------ Dereferenced *x[i] ----------
            *x[0]=1
            *x[1]=4
            ------------------------------------------------
```
